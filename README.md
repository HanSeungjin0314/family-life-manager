# Together Life v19

공동/개인 지출 요약 계산 보정 버전입니다.

## v19 변경
- 공동 지출 계산을 거래/고정비 각각 분리해서 다시 계산
- 전체 지출 카드 추가
- 기존 데이터의 scope 값이 비어 있거나 한글/영문 혼합이어도 공동/개인 계산 보정
- 거래내역 조회 limit 제거
- Supabase DB 변경 없음


## v19.4 계산 기준

- 수입: 수입으로 선택된 거래만 합산합니다.
- 변동 지출: 고정비를 제외한 지출 거래만 합산합니다.
- 전체 지출: 변동 지출과 고정비를 모두 합산합니다.
- 공동 지출: 공동 지출 거래와 공동 고정비만 합산합니다.
- 개인 지출: 개인 지출 거래와 개인 고정비만 합산합니다.
- 남은 금액: 모든 계좌 잔액을 합산합니다.
- 거래 입력 선택사항에서 이체를 제거했습니다.
- Supabase DB 변경은 없습니다.
